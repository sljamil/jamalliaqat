PRO_01 // AI + Embedded · GitHub
Duino.ai — Arduino Controlled by AI
Control Arduino hardware using natural language AI prompts through an interactive web-based GUI dashboard.
Arduino / C++HTML / JSAI APINode.js
DETAILS▼
Problem
Controlling Arduino hardware typically requires writing and uploading code for each new task. This project removes that barrier entirely by allowing natural language commands via AI.
Approach
Built a relay server in Node.js that passes user prompts to an AI API. The AI generates Arduino commands which are sent over serial to the microcontroller. A web-based GUI dashboard provides the interface.
Tech Stack
Arduino IDE (C++), HTML/CSS/JS dashboard, Node.js relay server, AI API integration, serial communication.
Outcome
Working prototype that lets users interact with physical hardware through conversational AI. Published on GitHub as open source.

🎓FINAL YEAR PROJECT// NUST · College of Aeronautical Engineering
FYP_01 // Sep 2022 – Sep 2023
Non-Invasive Glucose Monitoring Using Near-Infrared Spectroscopy
NIR spectroscopy-based glucose sensor prototype tested on aqueous solutions, fruit samples, and human subjects non-invasively. Supervised by Dr. Adnan Shah & Dr. Waqar Ahmad.
NIR SpectroscopySignal ProcessingMATLABOptical SensingEmbedded
FULL DETAILS▼
Problem
Current glucose monitoring requires blood draws (finger pricks), causing discomfort and infection risk. A non-invasive optical alternative would transform diabetes management globally.
Approach
Used Near-Infrared (NIR) light to illuminate tissue. Different glucose concentrations absorb NIR at characteristic wavelengths. A photodetector captures the transmitted/reflected light and MATLAB processes the spectral data to predict glucose concentration using regression models.
Testing
Validated on three progressive stages: aqueous glucose solutions (ground truth), fruit samples (biological matrix), and non-invasive human fingertip measurements.
Tech Stack
NIR LED array, photodetector, transimpedance amplifier, ADC, MATLAB (signal processing + regression modelling), custom optical housing.
Outcome
Working prototype demonstrating correlation between NIR absorbance and glucose concentration. Presented and approved as BS final year project at NUST CAE under dual supervision.
